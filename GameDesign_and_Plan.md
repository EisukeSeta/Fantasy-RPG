# 🌸 平安魔道伝 (Heian Mado-Den) - ゲーム設計書 Ver. 1.1

## 1. 世界観と背景 (Lore)
時は平安。都は闇に飲まれ、邪悪な魔物が跋扈している。
毎夜、黒煙と共に響き渡る不気味な「鵺（ぬえ）」の鳴き声が、帝を病に伏させ、民は度重なる不幸に見舞われていた。
薬や祈祷をもってしても効果は無く、源頼政より命を受けた一団は、怨念の核である鵺を調伏するため、迷宮「平安の闇」へと足を踏み入れる。

## 2. 登場人物と職種 (Characters & Classes)

### ⚔️ 武者 (Musha) - 渡辺 綱
- **特性**: 常人離れした剣技を持つ前衛の要。物理攻撃の命中・ダメージに優れる。
- **成長**: 初期マナ(Mana/MP)は 0 だが、レベルアップに伴い「精神の練度」が目覚め、マナが覚醒する。
- **奥義**: 一定レベルに達すると「蜘蛛切り」などの武技をマナを消費して放てるようになる。

### ☯️ 陰陽師 (Onmyoji) - 安倍 晴明
- **特性**: 呪法を操る攻撃の主。物理攻撃は弱いが、属性ダメージを与える強力な術を放つ。
- **成長**: マナとマナの成長率が高く、複数の術を使い分ける知識（Lv要求）を持つ。

### 📿 尼僧 (Nisou) - 八百比丘尼
- **特性**: 慈悲の心で仲間を癒やす支援者。不老不死の伝説に基づき、高い生存力を持つ。
- **成長**: 回復呪法と、守備（AC）を高める支援術に秀でる。

## 3. ゲームシステム (Mechanics)

### 📈 成長システム (S-Curve Progression)
- **必要経験値**: 成長はS字カーブを描き、Lv2へは 100 程度の経験値（初期の敵 2〜3 体撃破）で迅速に到達する。
- **レベル上限**: 第一段階の飽和点は Lv50 と設定。
- **特典**: レベルアップ時に HP/マナ（MP）が全回復し、最大値が上昇する。最大値上昇はランダム要素を含む（職種により傾向が異なる）。

### 👹 戦闘システム (Battle)
- **命中計算**: 攻撃側のACと防御側のACを比較して算出。ACが低い（平安の鎧が堅い）ほど攻撃が当たりにくくなり、ダメージも軽減される。
- **マナ（MP）**: 術や奥義の行使に不可欠。武者もレベル2以降で習得し、戦略の幅が広がる。
- **逃走 (Run)**: プレイヤー平均Lvと敵Lvを比較して成功判定。失敗時は「背後を突かれた」として一方的な敵の猛攻を受ける。
- **討死 (Death)**: HPが 0 になると行動不能（コマンドスキップ）。全滅すると「DEAD」となる。

### ⛩ フィールドギミック (Gimmicks)
- **神社の結界 (⛩)**: 立ち寄ると HP/マナが全回復し、討死した仲間が「神仏の慈悲」で復活（蘇生）する。
- **古の巻物 (📜)**: 特定の座標(1,2)に落ちており、都の背景やミッション（プロローグ）を伝える。
- **出口 (✨)**: 迷宮の主を倒した者のみに現れる聖域。踏むことで階層踏破の選択が可能。

## 4. 魔物ラインナップ (Bestiary)
平安の闇に潜む怨霊たち（一部抜粋）：
1. **餓鬼**: 序盤の雑魚。
2. **骸骨**: 物理攻撃の練習台。
3. **鉄鼠 / 人魂**: マナや魔法の耐性を試す存在。
4. **輪入道 / 青坊主**: 中レベル帯の強敵。
5. **土蜘蛛 / 牛鬼**: 高レベルの殺戮者。
6. **鵺 (Nue)**: 迷宮の主。雷と黒煙を操る。

## 5. 今後の拡張計画 (Roadmap)
- **装備品の実装**: 迷宮内に落ちている太刀や数珠、御札によるステータス変化。
- **階層の深化**: より複雑な構造と、鍵のかかった「怨霊の扉」。
- **新規呪法**: 混乱や睡眠などの状態異常を伴う高度な陰陽術。
- **中ボス**: 第一階層を突破した先に待ち構える「酒呑童子」。
