# 平安魔道伝 ― 第二章：羅生門（月下の冥府律） v2.0.0 実装企画書・ロードマップ

現在の「v1.0.0（基本探索と和風ダイアログの完成）」を基盤とし、テーマである「業による異形化」と「テキストアドベンチャー並みの深いシナリオ表現」を実現するための段階的なアップデート計画です。

既存のシステムを壊さず、少しずつ拡張していくためのステップを5つのフェーズに分割しています。

---

## 🗡️ Phase 1: 叙事詩的ダイアログシステム（テキストアドベンチャー化）
現在の `activeDialog` システムを拡張し、シナリオ演出の幅を広げます。

1.  **話者の表示機能**:
    *   ダイアログのタイトル部分だけでなく、「誰が話しているか（綱、晴明、比丘尼、下人、魔物など）」の顔アイコンや名前を表示できるようにする。
2.  **没入型エフェクトの追加**:
    *   テキスト表示に合わせて、画面全体（`dialog-overlay` の背景など）に簡単なCSSアニメーションエフェクトをかけられるようにする。
    *   例: 画面が赤く染まる（血飛沫）、青く揺らぐ（水底）、ノイズが走る（狂気）。
3.  **イベント・フラグへの対応**:
    *   メッセージを最後まで読んだ際に、単に閉じるだけでなく「特定のフラグを立てる（イベント完了など）」ためのコールバック関数 (`onComplete`) を厳密に処理する仕組み。

---

## 👹 Phase 2: 「業（カルマ）」システムとレベルアップの分岐
本作の核となる、「強くなるほど人ではなくなる」システムの実装。

1.  **ステータス構造の拡張**:
    *   各キャラクターのステータスに `karma`（業値）と `form`（形態：人間、半妖、完全な魔など）を追加。
2.  **「人の道」か「魔の道」かの二者択一**:
    *   レベルアップ関数 (`handleLevelUp`) を改修。自動でレベルアップさせるのではなく、専用のダイアログを開きプレイヤーに選択を迫る。
    *   **人の道を選択**: ステータス上昇は控えめ。人間性を維持（カルマ変動なし）。
    *   **魔の道を選択**: ステータスが大幅に上昇（あるいは強力な新スキル解放）。しかしカルマ値が上昇し、顔アイコンや肩書き（例：「武者」→「修羅」）が不気味なものへと変化する。

---

## ⛩️ Phase 3: 冥府の三階層（多層化ダンジョンシステム）
現在の1枚マップから、最下層を目指す立体的な迷宮へ。

1.  **多層マップ構造の導入**:
    *   現在の `mapData` を3次元配列（または階層ごとのマップの配列 `[floor1, floor2, floor3]`）に変更。
    *   プレイヤーのステータスに `z`（現在の階層）を追加。
2.  **昇降ギミック（階段）の実装**:
    *   特定の座標にボスを配置し、討伐後に「下り階段 (▽)」が出現するギミックを追加。
3.  **階層別テーマカラーとマテリアル**:
    *   各階層（第一層：鵺、第二層：裏内裏、第三層：人魚）に応じて、WireframeView（3D描画部分）の壁の色や床のテクスチャカラーを動的に変更し、視覚的に深度を表現する。

---

## 📜 Phase 4: 語り部「下人」のUIアップデート
羅生門の下からプレイヤーを俯瞰するメタ的な存在の演出。

1.  **ログウィンドウ（Map & Log）の改修**:
    *   現在のシステムメッセージ（「〜のダメージ！」など）の合間に、語り部である「下人」の皮肉めいた独白を挿入する。
    *   ログのフォントや色を調整し、「誰かの手記」や「下人の語り」のように見せる。
2.  **動的BGM / 環境音の制御 (Option)**:
    *   探索時の没入感をさらに高めるため、階層やカルマの進行度に応じて、水滴の音、心音、不気味な風の音などをブラウザの Audio API で静かに再生する基盤を用意する。

---

## 🎭 Phase 5: マルチエンディングとシナリオ統合
最終フェーズ。作成したマスターシナリオをゲーム内に全て組み込む。

1.  **道標（立て札）の全階層配置**:
    *   新マップの各所に、狂気に満ちた立て札や、過去の冒険者の手記を配置し、世界観の謎解き要素を入れる。
2.  **エンディング分岐ロジック**:
    *   第三層（最下層）の最終ボス討伐時、パーティー全員の総カルマ値を計算。
    *   カルマ値が一定以下なら「暁の奪還（ノーマル/トゥルー）」、一定以上なら「冥府の戴冠（ダークエンド）」の専用長文ダイアログ（プロローグと同等の豪華な演出）を再生してクリア画面へ移行させる。
3.  **全体のバランス調整とデバッグ**:
    *   敵の強さ、成長カーブの再調整、表示バグの潰し込み。
