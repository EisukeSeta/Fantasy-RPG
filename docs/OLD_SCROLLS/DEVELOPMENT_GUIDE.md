# 🚀 Fantasy-RPG 開発環境ガイド

このプロジェクトには、Antigravity の **Customizations (Rules & Workflows)** 機能を活用した、プロフェッショナルな自律開発環境が構築されています。

## 🛠️ 主な機能

### 1. プロジェクト固有のルール (`.gemini/rules/project-rules.md`)
- AIが「シニアフロントエンドエンジニア」として、高品質なコードを生成します。
- **言語一貫性**: すべての応答、コードコメント、コミットメッセージは日本語で行われます。
- **技術スタック**: React 19, Vite, Vanilla CSS に特化したコーディング。
- **自己修復**: 生成後のエラー（ビルド、リンター）を自ら検知して修正します。

### 2. オートメーション・ワークフロー (`.agents/workflows/auto-dev.md`)
以下のコマンドを使用することで、開発サイクルを自動化できます：

> 💬 **「/auto-dev [作成したい機能の依頼]」**

このコマンドにより、AIは以下のステップを自動で実行します：
1. **調査**: 既存のソースコードと開発ログを読み込む
2. **設計・記録**: `docs/PLANNING_LOG.md` に設計案と意思決定を追記
3. **実装**: 分割せず、一発で動作する完全なコードを出力
4. **検証・修復**: `npm run lint` / `npm run build` でエラーがあれば自己修復
5. **コミット**: 全て正常であれば、適切なプレフィックスで Git コミット

### 3. 意思決定の継続性 (`docs/PLANNING_LOG.md`)
全てのタスク開始前にこのログが読み書きされるため、AIが変わっても「過去に何が決まったか」を引き継ぐことができます。

---

## 📱 スマホ・PCでの実機確認フロー
1. AIがコード生成 → 自動テスト → コミットを完了します。
2. ユーザーはローカル環境（`npm run dev`）で確認するか、GitHub に Push します（`git push`）。
3. GitHub Actions により `deploy.yml` が走り、GitHub Pages 等へ自動デプロイされます。
4. 全デバイスのブラウザから最新の成果物を確認できます。
