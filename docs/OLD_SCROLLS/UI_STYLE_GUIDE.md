# ⛩️ 平安魔道伝 羅生門編：意匠・実装ガイドライン (UI_STYLE_GUIDE.md)

都の美しさを保ち、継承するためのデザインシステムと実装方針。

## 1. 色彩の法典（カラーシステム）
都を構成する基本の色。

| 変数名 | 色 | 用途 |
| :--- | :--- | :--- |
| `--primary-gold` | #b89a42 | ウィンドウ枠、タイトル、重要なアクセント。 |
| `--soft-gold` | #f0e68c | Lv表示、特別な文字、装飾。 |
| `--dark-red` | #500e0e | 各種バーのベース、暗い情熱。 |
| `--ancient-black`| #0a0a0a | 背景、基盤、漆黒。 |

## 2. 言霊の色使い（メッセージログ）
ログに表示される文字の色は、その意味によって厳格に分類する。

- **`msg-normal` (白/灰)**: 通常の行動、探索、淡々とした事実。
- **`msg-damage_party` (薄紅/赤)**: 隊員の負傷、痛打、敵の猛攻。
- **`msg-heal` (緑/白)**: 回復、癒やし、静謐、心身の充填。
- **`msg-level_up` (黄/金)**: 昇格（レベルアップ）、経験値取得、特別な成果。
- **`msg-event` (白/太字)**: 接敵、重要な発見、都の変異（`addMessage` で付与）。

## 3. 三面鏡の法（各ウィンドウの責務）
画面は左、中央、右の三つの領域で構成され、奥行きのある世界（没入感）を演出する。

### A. 隊員之証 (Pane: Status)
- **役割**: プレイヤー側の状態管理。
- **表示内容**: 肖像、姓名、Lv、各種バー（HP/MP/EXP）。
- **演出**: 行動中の隊員は `.active-battler` クラスにより、黄金のオーラと右方への微動（`transform`）を与える。

### B. 都の景色 (Pane: Main)
- **役割**: 視界の提供と操作。
- **上部（ViewWindow）**: 3Dワイヤーフレームによる迷宮の視界。
- **戦闘時演出**:
    - 怪異（敵）の姿は中央に。
    - 御名のフォント（`.enemy-name`）は巨大（2rem〜）に保つ。
    - ボスの場合：`.boss-aura` による黄金の背景効果と、`*** 鵺 ***` などの特殊装飾を施す。
- **下部（Controls）**: 前進、後退、打ちかかる等のボタンを配置。

### C. 言霊の記録 & 迷宮図 (Pane: Log/Map)
- **役割**: 履歴と空間情報の掌握。
- **迷宮図**: 石壁は `#444`（visited）または `#111`（通常）で描き分け、視認性を維持する。
- **プレイヤーアイコン**: `▲` 記号を用い、向いている方向（`playerState.dir`）に合わせて回転させる。

## 4. 実装の掟（共通ルール）
- **退行の防止**: 新たな機能を追加する際も、ウィンドウタイトルの `overflow: visible` 設定を崩さず、タイトルの視認性を死守すること。
- **平安の言葉選び**: 「HP/MP」「全回復」といったメタ的な表現を避け、「生命/霊力」「癒やし」「心身が癒やされ、生命と霊力が満ちた」等の平安の世界観に即した言霊を用いること。
- **応答性（レスポンシブ）**: モバイル環境（`isForceMobile`）では、三つの領域を切り替え（`.mobile-active-pane`）で表現し、情報の密度を守ること。
