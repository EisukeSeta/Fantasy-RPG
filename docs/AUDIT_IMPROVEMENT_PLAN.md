# ⛩️ システム監査及び改善計画案 (AUDIT_IMPROVEMENT_PLAN.md)

都の現状を「プレミアム開発基盤」に照らし合わせ、さらなる高みへ昇華させるための計画。

## 1. 言霊の純化（Scenario.json への完全統合）
`App.jsx` 内に残る以下のハードコードされた言霊を外部化する。

- **戦闘行為**: 「打ちかかり」「痛打」「かわした」「霊力が足りぬ」
- **遭遇演出**: 「強大なる怪異の気配……」「闇から這い出た」「震撼させる古の怪異【鵺】」
- **固定UI**: 「記録」「奏曲：停」「隊員」「迷宮図」などのボタン文字。

## 2. 意匠の洗練（CSS の一元化と細部修正）
- **デバッグUI**: インラインスタイルを `index.css` のクラス（`.debug-panel` など）に移行。
- **ブラウザ部品の隠蔽**: スクロールバー（scrollbar-width, scrollbar-color）を都の色彩（黄金/黒）に強制上書きし、OSの標準ボタンを排除。
- **ボタン統一**: `dialog-btn`, `move-btn` などの派生クラスを整理し、聖典に沿った意匠（黄金の枠、墨引きの影）へ統合。

## 3. 智の進化（高度なデバッグ操作）
- **任意転移の法**: 「跳躍」ボタン押下時に、(x, y) を入力できるダイアログ（`prompt` 等）を実装。
- **因縁の操作**: ボス討伐フラグ（`bossDefeated`）の ON/OFF 制御ボタンの追加。
