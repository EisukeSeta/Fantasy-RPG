# 🏯 平安魔道伝：UI master 仕様書 (UI_MASTER_SPEC.md)

本文書は、都の「理（プログラム）」と「器（デザイン）」を調和させ、PC・モバイル双方で至高の没入感を提供するための、唯一無二の UI デザイン聖典である。

---

## 🎨 デザインの理（Visual Logic）
- **平安没入感**: 背景は漆黒（`#000`）、文字は黄金（`#f0e68c`）・深紅（`#8b0000`）を基調とする。
- **色彩設計**:
  - `varGold`: `#f0e68c` (主たる文字色)
  - `varRed`: `#8b0000` (警告・ダメージ)
  - `varDeepAsh`: `#1a1a1a` (枠組み・背景)
- **情報の即時性**: 現在の状況（視界・ステータス・ログ）を極力スクロールなしで視認可能にする。

---

## 🖥️ PC版レイアウト (三連の計)
PC版は広い画面を活かし、三つの「板（Pane）」を横に並べる。

| ステータス (左) | 迷宮主画面 (中) | 操作・記録 (右) |
| :--- | :--- | :--- |
| 隊員の位階・体力を常時表示 | 3D迷宮視界 / 戦闘画面 | 迷宮図(上) / 言行録(中) / 作戦指示(下) |

---

## 📱 モバイル版レイアウト (垂直の計)
モバイル版は画面高（Viewport）を最大限に活用し、垂直に「理」を積み上げる。
全体の高さは `100dvh` に固定し、スクロールなしで全要素を視認可能とする。

### 3.1 探索時 (EXPLORING)
1. **迷宮視界 (1:1正方形)**: 100vw の一等地に、十字キーを背景として配置。
2. **操作盤**: ミニマップ表示、隊員詳細への切り替えボタン。
3. **言行録**: 画面下端の残余領域をすべて活用。

---

## 🛠️ 実装の掟
1. **Responsive Design**: `flexbox` と `grid` を使い分け、`@media (max-width: 768px)` で垂直レイアウトへ切り替える。
2. **Viewport Fix**: モバイル時は `overflow: hidden; touch-action: manipulation;` を徹底し、意図せぬズームやバウンスを防ぐ。
3. **Component Identity**:
    - `MessageLog`: 自動追従スクロール必須。
    - `MapOverlay`: モバイル時は全画面、PC時は操作パネルの一部。
