# ⛩️ 平安魔道伝 羅生門編：統一意匠聖典 (UI DESIGN GUIDELINE)

都の「景色（UI）」に一貫性を持たせ、没入感を維持するための理（ルール）をここに記す。

---

## 🌫️ 1. 全画面オーバーレイ（Yugen Modal）
重要な情報（図録、経典、ヘルプ等）を表示する際は、以下の構造を厳守せよ。
既存の `.yugen-modal-*` クラス（`index.css`）を用いること。

- **背景**: `rgba(0, 0, 0, 0.94)` 以上の暗転 ＋ `backdrop-filter: blur(12px)`
- **ヘッダー構造**:
  - 左側: 標題（`yugen-modal-title`）。アイコン ＋ 明朝体。
  - 右側: 閉じるボタン（`yugen-modal-close`）。文字は「閉じる」で統一。
- **コンテンツ領域**: `yugen-modal-content` クラスを使用し、縦スクロールを許可する。

## ✒️ 2. タイポグラフィ（文字の理）
文字の大きさや種類は「揺れ」を最小限にする。

- **見出し**: `1.4rem` 〜 `1.8rem`。黄金（`varGold`）色。
- **本文（説明書等）**: `1.1rem` (18px)。
- **注釈・フッター**: `0.85rem` 〜 `0.9rem`。灰色（`#888`）。
- **フォント**: `Sawarabi Mincho` (明朝体) を基本とする。

## 🎨 3. 色彩の理
- **主色 (Primary)**: `var(--primary-gold)` (#b89a42)。装飾、枠線、重要文字。
- **背景 (Background)**: `#000` 〜 `#0a0a0a`。漆黒。
- **強調**: `var(--soft-gold)` (#f0e68c)。名前や数値。

## ⚡ 4. 捷径（ショートカット）の挙動
- 全てのオーバーレイは、**「他のオーバーレイが開いている時に別の捷径が押されたら、現在の画面を閉じてから新しい画面を開く」** という排他的連動（Mutual Exclusion）を実装せよ。
- `App.jsx` の `closeOthers` ロジックを常に最新の画面一覧と同期させること。

---

## 🔱 5. 模範的な実装
- `ArchivesView.jsx`: 全画面かつ2カラム、画像拡大機能を備えた最良の模範。
- `SpellGrimoire.jsx`: タブ切り替えとテーブル表示を備えた模範。

この聖典に記された理に背く UI の追加は、都の美意識への叛逆と見なす。
