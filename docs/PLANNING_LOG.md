## 📜 開発の掟 (Fixed Rules)
1.  **[行動予約の理]**: `lastProcessedTurnRef` と `battleTurn` の照合を必須とする。
2.  **[最新の参照]**: 非同期演出およびタイマー内では、必ず `Ref.current` 経由で最新の状態を参照する。
3.  **[格式ある記録]**: 全ての中核処理は `Logger` を介して構造化ログに刻む。
4.  **[言霊の不変]**: `Scenario.json`, `Enemies.json` のプロパティ削除は、主殿の勅命なき限り厳禁とする。
