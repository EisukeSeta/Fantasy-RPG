# ⛩️ 平安魔道伝 羅生門編 開発史：都の洗練

## 🛠️ 現在の課題と設計（実装計画）

### 1. [窓の威厳] ウィンドウタイトルの欠け修復
- **現状**: `.pane-status` 等の親要素に `overflow: auto` が設定されているため、負の `top` を持つ `.window-title` が切り落とされている。
- **対策**: `.window` に十分な `margin-top` を確保し、`.window-title` が確実に視界に収まるよう CSS の配置を調整する。

### 2. [石壁の復権] マップの壁視認性向上
- **現状**: `.visited` クラスと `.wall` クラスが混在した際、壁の色調（#444）が正しく認識されていない可能性がある。
- **対策**: CSS で `.map-cell.wall.visited` の優先順位を上げ、石壁として強調表示する。

### 3. [怪異の咆哮] 敵UIの強化
- **現状**: 怪異の御名が小さく、存在感が乏しい。
- **対策**: 
    - 常に御名を巨大化（`fontSize: 2.2rem` 等）し、黄金の彩りを与える。
    - ボスの場合、名前を `*** ${name} ***` で装飾し、`boss-name` クラスの演出を極大化する。

### 4. [死の引導と転生] ゲームオーバーと過酷な復活ロジック
- **現状**: 「否」の選択肢が無反応。復活時のペナルティが甘い。
- **対策**:
    - **終焉**: 「否」選択時に「都は闇に飲まれた……」等の辞世の句（メッセージ）を表示し、ドラマチックに `GAMEOVER` 画面へ。
    - **はい（復活）**: 井戸へ転送される前に「黄泉の理を書き換える……」等のメッセージを挿入。
    - **復活の洗礼**:
        - 生命/霊力（HP/MP）を最大値に関わらず `1` に。
        - 次のLvまでの経験値を `0` にリセット。
        - 迷宮図（mapData）の全 `visited` を `false` に戻し、初期地点周辺のみ再点灯（恐怖の再来）。

## 🛠️ 次なる都の洗練（追加課題）

### 5. [智の効率化] デバッグモードの活用
- **現状**: 検証に時間がかかっている。
- **対策**: `?debug=true` 時のデバッグ機能を再点灯（過去の資産活用）し、移動・遭遇・状態の操作を迅速化。

### 6. [言霊の浄化] メタ表現の排除
- **現状**: 「HP/MP全回復」等の表記が都の世界観を乱している。
- **対策**: `MapEvents.json` の記述を「体力と魔力が静かに満ちた」等の平安にふさわしい表現へ統合。

### 7. [八百比丘尼の覚醒] AI戦闘の最後の一人問題
- **現状**: メンバーが一人（主に八百比丘尼）になった際、AI戦闘が停止する不具合。
- **対策**: `App.jsx` の `handleFight` 内にあるターン自動遷移ロジックを精査し、最後の一人でも自動的に打ちかかるよう論理を正す。

## 🕰️ 2026-04-05/06 ハイブリッド・レイアウト v2.0：都の遍在
- **課題**: モバイルUXの劇的向上と、PC版の威厳あるレイアウトの両立。
- **解決**: 
    - ダンジョン視界を 1:1 正方形、全幅固定へ変更し、探索の迫力を最大化。
    - 十字キー（D-pad）をダンジョン内に半透明オーバーレイ配置し、視線移動を最小限に。
    - 浮遊式ステータス・ダッシュボードを導入し、探索・戦闘中でも行動中の隊員（黄金の枠）を常時監視可能に。
    - 画面遷移にタブ切り替えUIとスライドアニメーションを採用し、アプリらしい滑らかな操作感を実現。
    - 迷宮図（マップ）にピンチ・ズーム機能を実装。
    - 戦闘時の視認性確保のため、コマンドパネルを画面下部へ移動。
    - 手動戦闘時でも全ての「術式」を選択・発動できる専用オーバーレイを構築。
    - 各種機能をダンジョン下に集約し、片手での快適な操作環境を構築。

## 🕰️ 2026-04-06 v2.1 - 「平安の響き」スマホUX極大化
- **AI戦闘ロジックの完遂**: ターン管理変数 `battleTurn` の導入により、最後の一人になってもAIが止まらず調伏を続ける「不退転の理」を実装。
- **言霊（メッセージ）の浄化**: 「Lvアップ」を「位階昇格」へ、「回復」を「生命の満ち」へ。平安時代の世界観に即した情緒的な表現へ一括置換。
- **スマホ版ログ装飾**: ログ背景に和紙テクスチャ、さわらび明朝体、黄金のハイライトを適用。上下に霞（フェード）を加え、没入感を向上。
- **レイアウト構造の最適化**: 1:1ダンジョン視界とログ領域が、高解像度スマホ画面で正しく共存するネスト構造へ修正。
