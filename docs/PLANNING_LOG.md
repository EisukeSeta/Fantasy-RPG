# ⛩️ 合戦律・再編計画：[合戦管制塔（Combat Control Tower）の構築]

主殿。これまでの「モグラ叩き」を断ち切り、将来の多敵出現にも耐えうる、強靭かつシンプルな合戦ロジックを再構築する「大工事」の計画を奉納いたします。

## 1. 根本思想：[一元管理と確定的遷移]
- **分散の廃止**: AI・敵攻撃・ターン更新を司る複数の `useEffect` を解体し、単一の **「合戦律（Combat Master Loop）」** へ統合します。
- **公式フェーズの導入**: 不安定なガード（Ref）を廃し、都の法として **`combatPhase`** を定義します。
- **拡張性の確保**: 自・敵ともに複数（配列）での管理を前提とした、シンプルなインデックス・ループへと移行します。

## 2. 合戦の位階（Combat Phase）
1. `READY`: 入力待機（主殿の英断を待つ）。
2. `ACTION`: 行動中（演出・計算・メッセージ表示）。入力は物理的に遮断。
3. `CHECK`: 生死判定・次走者決定。
4. `VICTORY` / `GAMEOVER`: 終焉演出。
