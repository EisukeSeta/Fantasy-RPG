# 平安魔道伝 - 生死の循環 (GAME_LOOP_DESIGN)

本典には、都における探索と戦闘、そして「死と再生」の理（ゲームループ）を記す。

## 1. 基本となる循環 (Core Loop)

都の探索は以下の「四つの相」を巡り続ける。

1. **探索 (Explore)**: 暗闇（迷宮）の中、一歩ずつ都（マップ）を視認・記録する。
2. **遭遇 (Encounter)**: 怪異との邂逅、あるいは霊威（イベント）の発動。
3. **戦闘 (Battle)**: 怪異を調伏し、経験（EXP）と証（勝利）を得る。
4. **昇格 (Level-up)**: 得た経験により魂（レベル）を高め、さらなる深淵へ挑む。

## 2. 黄泉還りの理 (Resurrection System)

都に備わる独自の再生の理。

- **終焉 (Wipeout)**: 戦闘、あるいは災いによって隊員全員の生命が尽きること。
- **転生 (Resurrect)**:
  - **再起の地**: 生命の代償を払い、**「黄泉の井戸 (0,0)」** にて目覚める。
  - **衰弱の状態**: 復活直後の **体力（HP）および魔力（MP）は「`1`」**。
- **聖域の快癒**:
  - 瀕死の身で地上を這い、**「御神木の社 (1,1)」** へ到達した瞬間、生命と霊力が完全に癒やされる。

### 設計思想
- 全滅を「終わり」とせず、**「代償を払った再起」** と定義する。
- 復活直後の瀕死状態で (0,0) から (1,1) へ歩む緊張感を、都の「死の重み」とする。

## 3. 都の最終調伏 (End-Game Goal)

都の均衡を乱す根源、**「鵺 (8,6)」** の討伐を以て一章の完結とする。

- **道程の過酷さ**: 
  - (1,1) から (8,6) までの最短距離は、怪異の犇めく深淵。
  - 平均 Lv5-6 程度まで、近傍での「経験稼ぎ」と「転生」を繰り返すことで道が開かれる。

## 4. 行軍の心得 (Exploration UX)

- **暗闇の恐怖**: 未踏の地（マップ）を歩く際の視認範囲（1マス）を生命線とする。
- **社の安堵**: 唯一の全快地点 (1,1) を拠点とし、徐々に探索範囲を広げる「平安のリズム」を重視する。

---
*都に死はあらず、ただ再構築あるのみ。主殿の指揮のもと、都の平安を取り戻せ。*
