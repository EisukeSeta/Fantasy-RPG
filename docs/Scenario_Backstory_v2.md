# 平安魔道伝 ― 第二章：羅生門（月下の冥府律） マスターシナリオ＆バックストーリー

## 1. コア・テーマ
**「強くなるほど、人ではなくなっていく」**

平安京の南端、打ち捨てられた「羅生門」。そこは今や現世と冥府を繋ぐ巨大な「蓋」として開き、無限の迷宮と化していた。
都を覆う疫病と黒煙を断つため、三人の英雄が冥府の迷宮へと身を投じる。
しかし、冥府の魔を打ち倒し、その業（経験値や力）を取り込むことは、自らの肉体と精神を魔へと近づける行為でもあった。プレイヤーは「人の道」を保ちながら苦難の道を進むか、「魔の道」に堕ちて圧倒的な力を振るうか、究極の選択を迫られる。

---

## 2. 登場人物と「業（カルマ）」の変容

主人公となる三人の討伐隊は、それぞれ深い罪と「業」を背負っている。冥府の瘴気を吸い、魔物を討つほどに、彼らの隠された本性が視覚的にも精神的にも顕現していく。

### ⚔️ 渡辺 綱（わたなべ の つな）
*   **役割**: 前衛・物理アタッカー
*   **業のテーマ**: 豪勇と「殺生」の咎
*   **バックストーリー**: かつて一条戻橋で鬼（茨木童子）の腕を切り落とした豪傑。しかし、その腕は彼自身の腕に呪いとして融合しつつある。都を守るために振るう刃が、実は彼の中の「殺戮への渇望」を満たすためのものではないかと苦悩している。
*   **【魔の道】への変容**: 肌に硬い鬼の鱗が現れ、兜の下から角が突き出す。融合した茨木童子の腕が脈動し、会話テキストでも次第に「血」と「闘争」を求める狂戦士の口調へと変貌していく。UI画面には時折、血飛沫の幻影が重なる。

### ☯️ 安倍 晴明（あべ の せいめい）
*   **役割**: 後衛・呪法アタッカー
*   **業のテーマ**: 探求と「禁忌」の咎
*   **バックストーリー**: 類まれなる呪力と知識を持つ天才肌の陰陽師。白狐・葛の葉を母に持つという出自に対する強烈なコンプレックスと、人間の理解を超えた「深淵の真理」を知りたいという危うい知的好奇心を抱える。
*   **【魔の道】への変容**: 知識と魔力を得るほどに肉体が希薄化し、肌が透き通ってくる。瞳は人間性を失った金色の獣の眼となり、影は「狐」の形に固定される。会話は冷酷無比な超常存在となり、人間に対する感情を完全に喪失する。周囲に氷のごとき青い狐火が漂う。

### 📿 八百比丘尼（やおびくに）
*   **役割**: 中衛・回復と支援
*   **業のテーマ**: 慈愛と「不死」の咎
*   **バックストーリー**: かつて誤って人魚の肉を食らい、不老不死の呪いを受けた尼僧。数百年を生き、愛する者たちを看取り続けた結果、彼女の心は深海のように冷たく静まり返っている。「死」という救済を求めながらも、他者を癒やさずにはいられない矛盾を抱える。
*   **【魔の道】への変容**: 慈悲の象徴たる姿が、徐々に異形の人魚へと変貌。全身に青緑の鱗が広がり、体温を失う。表情は一切動かない「微睡む聖母像」のように固定され、狂気を孕んだ歪な母性で魔物すらも無意識に「抱擁（絞殺）」しようとする。水の滴る音響が付き纏う。

## 3. 羅生門・冥府の三階層

迷宮は大きく3つの階層（レイヤー）に分かれ、それぞれの階層が主人公個人の「業」と深く結びついている。各階層を降りるごとに、狂気の度合いが増していく。

### 第一層：鵺の階（ぬえのきざはし）
*   **テーマ**: 「血と闘争」 （渡辺綱の業と共鳴）
*   **環境**: 平安京の市街地が腐り、土塊と血肉に塗れたような悍ましい迷宮。至る所から獣の咆哮が響く。
*   **ボス**: **巨妖・鵺（ぬえ）**。都の疫病の源であり、幾多の武将の無念が寄り代となった大怨霊。
*   **演出**: 壁から血が滲み、戦闘のたびに「殺戮の悦び」を誘発するような狂的なメッセージがテキストに混じる。

### 第二層：裏内裏（うらだいり）
*   **テーマ**: 「狂気と禁忌」 （安倍晴明の業と共鳴）
*   **環境**: 帝の住まう内裏が鏡合わせのように反転し、無限に続く廊下と襖絵。幾何学的で狂った空間。空間自体がプレイヤーの正気を奪いに来る。
*   **ボス**: **金毛九尾・玉藻前（たまものまえ）**。幻術と狂気を操る大妖。
*   **演出**: 青白い月光の下、UIがグリッチを起こす、数式や呪符が画面を覆うなどのサイケデリックな演出。

### 第三層（最下層）：人魚の揺り籠
*   **テーマ**: 「停滞と永遠」 （八百比丘尼の業と共鳴）
*   **環境**: 深海のように暗く、水没した枯山水。永遠に時間が止まったかのような静寂。
*   **ボス**: **海座頭（うみざとう）** または **水底の古神**。
*   **演出**: 歩くたびに水音が響き、視界が歪む。全てを諦め、永遠の眠りにつくことを促す甘い声がログウィンドウに囁き続ける。

## 4. 特殊演出「語り部：下人（げにん）」

本作のナビゲーターであり、システムメッセージ全般を担当するメタキャラクター。
芥川龍之介の「羅生門」に登場する下人のように、羅生門の陰にしゃがみこみ、プレイヤーの戦いを皮肉交じりに見下ろしている。
（例：ログウィンドウにて、「ほう、見事な太刀筋だ。だが、お前さんの眼も随分と鬼に似てきたな…」等と語りかけてくる）

## 5. マルチエンディング構想

プレイヤーのレベルアップ時の選択（人の道 / 魔の道）と、カルマ値の集計によって、最下層で待ち受ける真の結末が変化する。

1.  **「暁の奪還」エンディング（人の道）**：
    凄絶な苦労の末、人間性を保ったまま冥府の蓋を閉じる。三人はボロボロになりながらも都に朝焼けを取り戻す。「英雄」としての確かな終幕。難易度は極めて高い。
2.  **「冥府の戴冠」エンディング（魔の道）**：
    強さを求め異形化しきった三人は、羅生門の深淵を支配する。平安京は魔界に飲み込まれ、三人は新たな「冥府の王」として永遠に君臨するダークエンド。無双の爽快感の果ての虚無。
3.  **「業の輪廻」エンディング（中途半端）**：
    迷いの中、魔の力に飲み込まれ、羅生門の一部として石化する。そしてまた新たな討伐隊がやってくる無限ループ。
