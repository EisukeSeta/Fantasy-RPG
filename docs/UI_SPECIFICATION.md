# 平安魔道伝 羅生門編 UI意匠仕様書 (v1.0)

本文書は、都の「理（プログラム）」と「器（デザイン）」を調和させ、PC・モバイル双方で一貫した操作環境を提供するための仕様を定義する。

## 1. デザイン原則
- **平安没入感**: 背景は漆黒、文字は黄金・深紅を基調とし、和の情緒を尊ぶ。
- **情報の即時性**: 現在の状況（視界・ステータス・ログ）を極力スクロールなしで視認可能にする。
- **一貫性**: PCとモバイルで共通のコンポーネントを用いつつ、配置のみを最適化する。

## 2. PC版レイアウト (3連の計)
PC版は広い画面を活かし、三つの「板（Pane）」を横に並べる。

| ステータス (左) | 迷宮主画面 (中) | 操作・記録 (右) |
| :--- | :--- | :--- |
| 隊員の位階・体力を常時表示 | 3D迷宮視界 / 戦闘画面 | 迷宮図(上) / 言行録(中) / 作戦指示(下) |

- **全画面固定**: `overflow: hidden` により、ブラウザ全体のスクロールは発生させない。
- **言行録**: 固定の高さを持ち、内部でスクロール可能。

## 3. モバイル版レイアウト (垂直の計)
モバイル版は画面高（Viewport）を最大限に活用し、垂直に「理」を積み上げる。
全体の高さは `100vh` (または `100dvh`) に固定し、スクロールなしで全要素を視認可能とする。

### 3.1 探索時 (EXPLORING)
1. **迷宮視界 (1:1正方形 / 100vw)**:
   - 横幅いっぱいに広げ、高さも横幅と等しく（100vw）保つことで 1:1 のアスペクト比を維持する。
   - 十字キーは透明なオーバーレイとして画面上に配置。
   - 左上にミニステータスを浮遊表示。
2. **操作盤 (flex: 0 0 auto)**:
   - 迷宮図、隊員証、記録、音響の4ボタンを配置。
3. **言行録 (flex: 1 / 残余領域)**:
   - 画面の下端まで、残された領域をすべて活用する。

### 3.2 戦闘時 (BATTLE)
1. **戦場視界 (1:1正方形 / 100vw)**:
   - 探索時と同様に 1:1 を維持し、大型の敵画像も歪みなく表示する。
2. **軍略指令 (flex: 0 0 auto)**:
   - コマンドボタン群。
3. **言行録 (flex: 1 / 残余領域)**:
   - 戦闘ログを表示。

## 4. コンポーネント定義
- **MessageLog**: 
  - 自動最下部スクロール機能を備える。
  - 親要素の残余スペース（`flex: 1`）をすべて使い、内部スクロール可能。
- **ControlPanel**:
  - モバイル時は冗長な移動ボタン（Arrows）を非表示化し、コマンドのみに注力。
- **LabyrinthMap / StatusPane**:
  - モバイル時はボタン押下時に全画面オーバーレイ（`fixed`）として出現する。

## 5. 実装への申し送り事項
- `game-container` は `height: 100vh` と `display: flex; flex-direction: column` を維持すること。
- モバイル版の `pane-main` 等の高さは、`px` 指定ではなく `vh` を用いて、あらゆる画面サイズに適合させること。
