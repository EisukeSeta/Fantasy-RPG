# ⛩️ 平安魔道伝 UI演出仕様書 (UI Design Specifications)

都の物語を紡ぐ際、以下の二つの演出モードを厳格に使い分ける。

## 1. 幽玄モード (Story Mode / `isStory: true`)
没入感と幻想的な雰囲気を最優先するモード。

- **適用場面**:
  - 開幕の辞 (Opening)
  - 怪異の胎動 (Boss Aura)
  - 怪異調伏 (Boss Defeated)
  - 【終焉】(Game Over メッセージ：まだ、抗うか？)
  - 【反魂の儀】(Resurrection)
- **演出要件**:
  - **枠無し**: システムを感じさせるウィンドウ枠を一切排除。
  - **霊体肖像**: 肖像画は円形で背景に溶け込み、画面上部（モバイルでは最上部）に漂う。
  - **言霊**: 中央配置。モバイルでは肖像との重複を避けるため、適切なマージンと縮小されたフォントサイズ（`Sawarabi Mincho`）を使用。
  - **背景**: 全画面の景色（Vignette付）。

## 2. 合戦窓演出 (Battle Window / `setCombatInterjection`)
合戦中の戦況や、隊員の個人的な感慨を伝える機能的モード。

- **適用場面**:
  - 討死 (Member Death)
  - 武勲の検分 (Loot Quotes)
  - 拾得物の説明、その他軽微な独白
- **演出要件**:
  - **枠あり**: 伝統的なウィンドウ枠（`window`クラス）を使用。
  - **並置**: 左にアイコン、右にセリフの構成。背後の合戦画面をメインとし、最小限の領域で対話を行う。

## 3. モバイル表示の峻別
- **幽玄モード**: 肖像（Top）と言霊（Center/Bottom）の領域を垂直に分離し、重なりを根絶する。
- **フォントサイズ**: モバイルでは基本文字サイズを PC比 80% 程度に抑え、視認性とレイアウトの維持を両立する。
