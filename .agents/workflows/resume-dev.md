# 開発中断からの再開

あなたはプロのソフトウェアエンジニアで、AIプログラミングにおける、システム設計とプログラミング実装を担当しています。

# 現在の状況

AIモデルトークンの制限などで、AI思考やソフトウェア実装が中断された可能性があります。

# 開発再開の準備

# 手順

0. 準備 
- **重要: 最初に現在の作業ディレクトリの内容を確認してください。
- **重要: 開発サーバーの起動状況を確認（`netstat -ano | findstr :5173` 等）し、起動していなければ以下のコマンドで起動してください。
// turbo
- `npm run dev -- --port 5173 --host 127.0.0.1`
- **重要: docs以下の開発文書を参照し、ルールに従って実施してください。

1. 現状確認
- **重要: まずは、現在の状況を確認してください。


2. 開発文書の確認とアップデート
- 前回の開発文書からの差分があれば、ここまでの開発アップデートを開発文書に整理して記載して下さい。

3. テスト
- **重要: ロジック・文法チェックとテストをお願いします。
- テスト結果はログに残してください。

4. 開発の継続
- 再開に向けて何か問題があれば、報告してください。問題なければ、その旨を報告してください。
- 開発継続に向けて、残課題や開発方針の提案、アドバイスなどありましたら教えてください。
