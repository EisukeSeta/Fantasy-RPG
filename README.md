# 🏮 平安魔道伝 (Heian Mado-Den)

平安時代の京都を舞台にした、一人称視点（DRPG）のダークファンタジーRPGです。
怨念に満ちた迷宮を探索し、都を病に陥れる伝説の魔物「鵺（ぬえ）」を調伏してください。

## 🎮 プレイ方法
1. **探索**: `W / A / S / D` または `矢印キー` で迷宮を移動します。
2. **戦闘**: 魔物に出会うと「打ちかかる」「術・祈祷」「逃げる」から行動を選択します。
3. **成長**: 魔物を倒して「功徳（経験値）」を積み、レベルアップで能力を高めます。
4. **守護**: ⛩（神社の結界）に立ち寄ることで、HP/マナが回復し、討死した仲間が復活します。
5. **物語**: 📜（古の巻物）を見つけ、都の危難と己の使命を確認してください。

## ⚔️ 主要な登場人物
- **渡辺 綱 (武者)**: 剣心の極致、マナの目覚めと共に奥義を放つ。
- **安倍 晴明 (陰陽師)**: 呪法の大家、式神と属性の術で魔を滅ぼす。
- **八百比丘尼 (尼僧)**: 慈悲の化身、仲間の傷を癒やし守りを固める。

## 👹 闇に潜む魔物たち
餓鬼、骸骨、鉄鼠、人魂、輪入道、濡れ女、青坊主、土蜘蛛、牛鬼、そして階層の主「鵺」。
