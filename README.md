# Fantasy RPG - Labyrinth of the Beginnings

初期の「Wizardry」にインスパイアされた、ブラウザで動作するレトロな1人称視点3Dダンジョン探索RPGです。

## 🎮 特徴

*   **1人称視点3Dビュー**: SVG描画によるパースペクティブ付きのワイヤーフレームダンジョン。
*   **オートマッピング**: 探索したエリアが右側のマップウィンドウに自動で記録されます。
*   **ターン制バトル**: AC（アーマークラス）やダイスロールに基づいた古典的な戦闘システム。
*   **レトロなUI**: 日本語ピクセルフォント（DotGothic16）を採用した黒背景の硬派なデザイン。

## ⌨️ 操作方法

*   **W / ↑**: 前進 (Forward)
*   **S / ↓**: 後退 (Backward)
*   **A / ←**: 左を向く (Turn Left)
*   **D / →**: 右を向く (Turn Right)
*   **戦闘時**: 画面右側のコマンドボタン（たたかう / にげる）をクリックして行動。
*   **デバッグ**: メインビューをクリックすることで、敵とのエンカウントを手動で切り替えることができます。

## 🛠 技術スタック

*   **Frontend**: React (Vite)
*   **Rendering**: SVG (for 3D wireframe), CSS Grid (for UI layout)
*   **Styling**: Vanilla CSS
