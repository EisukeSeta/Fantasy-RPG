# 🏮 平安魔道伝 (Heian Mado-Den)
**v4.9.7 "統治の礎" - Test-Driven RPG Development Project**

平安時代の京都を舞台にした、一人称視点（DRPG）のダークファンタジーRPGです。
怨念に満ちた迷宮を探索し、都を病に陥れる伝説の魔物「鵺（ぬえ）」を調伏してください。

---

## 🎮 プレイ方法
1. **探索**: `W / A / S / D` または `矢印キー` で迷宮を移動します。
2. **戦闘**: 魔物に出会うとオートバトルが開始されます。戦況に応じた術の発動、または手動での指揮も可能です。
3. **成長**: 敵を倒して「功徳（経験値）」を積み、レベルアップで能力を高めます。本作はシグモイド曲線に基づいた重厚な成長体験を提供します。
4. **再起**: 全滅時は「黄泉の井戸」から御神木の社へと帰還します。社の安らぎの中で体力を全快し、再び迷宮へと挑んでください。

---

## ⚔️ 主要な登場人物 (Contract-Guaranteed Stats)
- **渡辺 綱 (武者)**: 高い生命力と打撃力を誇る都の守護者。
- **安倍 晴明 (陰陽師)**: 強力な範囲攻撃術を操り、魔を掃滅する。
- **八百比丘尼 (尼僧)**: 慈悲の術で仲間を癒やし、防御の結界を張る。

## 🎌 プロジェクトの特長：AI共生開発モデル
本作は、**「AIエージェントと人間の高度な共鳴」**によって開発されています。単なるコード生成ではなく、以下の厳格な「理（プロセス）」によって品質が担保されています。

### 1. テストファースト統治 (Test-Driven)
全ての合戦ロジック（命中・ダメージ・痛打計算）や成長ロジックは、**Vitest** によるユニットテストで常時監視されています。AIがコードを改変する際、既存のバランスを破壊することを論理的に防いでいます。

### 2. 三権分立の掟 (Roles)
AIエージェントに「設計者」「実装者」「監査官」の三役を演じ分けさせ、実装前の設計Artifact提示と、実装後の厳格な削除チェックを徹底しています。

### 3. バランステスト・シミュレーション
Node.js上で動作する独自の **「バランステスター」** を備えています。1000回以上の合戦を瞬時にシミュレートし、常に「雅（Normal）」な難易度バランスが維持されていることを科学的に計測しています。
